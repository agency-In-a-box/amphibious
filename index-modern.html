Why developers choose Amphibious

Everything you need to build beautiful, responsive websites without the bloat

Modern Grid System

16-column flexbox grid with responsive breakpoints. No floats, no clearfixes, just clean modern CSS.

Dark Mode Ready

CSS custom properties make theming a breeze. Switch between light and dark modes with a single class.

25+ Components

Buttons, cards, modals, navigation, forms, and more. All built with semantic HTML and accessibility in mind.

Lightning Fast

Under 50kb gzipped. No JavaScript required for styles. Tree-shakable when you need only specific components.

Accessibility First

WCAG 2.1 compliant. Semantic HTML, ARIA attributes, keyboard navigation, and screen reader support.

E-commerce Ready

Product cards, shopping carts, checkout flows, and rating systems. Perfect for online stores.
