### CSS Classes
- `.amp-nav` - Main navigation wrapper
- `.amp-nav--sticky` - Sticky on scroll

- `.amp-nav__menu--open` - Mobile menu open state
- `.amp-footer` - Footer wrapper
- `.amp-footer--minimal` - Minimal footer variant
