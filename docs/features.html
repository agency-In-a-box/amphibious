JavaScript Implementation

let slideIndex = 1;

function changeSlide(n) {
  showSlide(slideIndex += n);
}

function currentSlide(n) {
  showSlide(slideIndex = n);
}

function showSlide(n) {
  const slides = document.querySelectorAll('.slide');
  const dots = document.querySelectorAll('.slide-dot');

  if (n > slides.length) slideIndex = 1;
  if (n < 1) slideIndex = slides.length;

  slides.forEach(slide => slide.classList.remove('active'));
  dots.forEach(dot => dot.classList.remove('active'));

  slides[slideIndex - 1].classList.add('active');
  dots[slideIndex - 1].classList.add('active');
}

Amphibious 2.0 is designed to work seamlessly with the Lucide icon library. You can find the complete icon set and usage instructions at lucide.dev.

// Install Lucide via npm
npm install lucide

// Or include via CDN
<script src="https://unpkg.com/lucide@latest/dist/umd/lucide.js"></script>

// Initialize icons
lucide.createIcons();

Best Practices

Performance Optimization

  • Use CSS transforms for slideshow transitions instead of changing layout properties
  • Lazy load images in slideshows to improve initial page load times
  • Minimize the number of utility classes per element for cleaner HTML
  • Use SVG icons instead of icon fonts for better performance and accessibility

Accessibility Guidelines

  • Always provide alt text for slideshow images
  • Include keyboard navigation for slideshow controls
  • Use semantic HTML elements with utility classes rather than purely presentational divs
  • Ensure sufficient color contrast when using colored text utilities
  • Provide meaningful labels for icon-only buttons

Maintainability

  • Create custom CSS classes for repeated utility combinations
  • Use consistent spacing scales throughout your project
  • Document custom component variants and their use cases
  • Consider creating a design system for complex applications
