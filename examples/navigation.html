Navigation States

Sticky Navigation

Add .nav-sticky class to make navigation stick to top when scrolling

Fixed Navigation

Add .nav-fixed class for fixed positioning at top of viewport

Transparent Navigation

Add .nav-transparent class for transparent navigation (useful over hero images)

<nav class="nav-sticky">...</nav>
<nav class="nav-fixed">...</nav>
<nav class="nav-transparent">...</nav>

Responsive Behavior

Navigation automatically adapts to different screen sizes:

  • Desktop (>800px): Full horizontal navigation with dropdowns
  • Tablet/Mobile (≤800px): Navigation collapses, toggle button appears
  • Mobile Portrait (≤480px): Optimized padding and font sizes

To hide navigation on mobile and show toggle button:

<button class="nav-toggle">...</button>
<nav class="nav-mobile-hide">
  <ul class="horizontal">...</ul>
</nav>

Accessibility Features

  • Skip navigation link for keyboard users
  • ARIA labels and expanded states
  • Focus visible indicators
  • Screen reader only text support
  • Reduced motion support
  • High contrast mode compatible
<!-- Skip navigation -->
<a href="#main-content" class="skip-nav">Skip to main content</a>

<!-- ARIA navigation -->
<nav aria-label="Main navigation">...</nav>

<!-- Screen reader text -->
<span class="sr-only">Current page</span>
