About Amphibious

Amphibious is a modern CSS framework that elegantly scales from desktop to mobile devices. Built on a familiar 16-column grid system, it provides the flexibility and power needed for responsive web design while maintaining simplicity and ease of use.

The framework includes a comprehensive set of components, utilities, and patterns that help developers build beautiful, accessible websites quickly. With its focus on semantic HTML and modern CSS features, Amphibious ensures your sites are future-proof and performant.

Building Accessible Interfaces

Accessibility isn't just about compliance‚Äîit's about ensuring everyone can use your website effectively. By following WCAG guidelines and using semantic HTML, you create experiences that work for all users, regardless of their abilities or the devices they use.

Key Principles
  • Perceivable: Information must be presentable in ways users can perceive
  • Operable: Interface components must be operable by all users
  • Understandable: Information and UI operation must be understandable
  • Robust: Content must be robust enough for various assistive technologies
Best Practices

Start with semantic HTML as your foundation. Use heading elements to create a logical document structure. Provide text alternatives for non-text content. Ensure sufficient color contrast. Make all interactive elements keyboard accessible. Test with screen readers and other assistive technologies.
