How it works

An atomic design system that scales

Amphibious uses Brad Frost's Atomic Design methodology. Start with design tokens, build atoms, combine into molecules, and compose complex organisms. Every component works together seamlessly.

  • Design tokens: colors, typography, spacing foundation
  • Atoms: buttons, badges, icons, spinners
  • Molecules: alerts, progress bars, tooltips
  • Organisms: cards, navigation, modals, forms
